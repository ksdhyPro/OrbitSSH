# 更新日志

## v1.1.5

1. 优化 SFTP 上传入口逻辑：无论当前是否选中文件或文件夹，右键菜单都会显示“上传文件”和“上传文件夹”，并统一上传到当前所在目录。
2. 优化文件夹上传流程：选择文件夹后先扫描子项并记录相对路径，再按目录、文件队列逐项创建或上传，任务中心以一个上传组展示整体进度。
3. 修复上传组暂停后继续可能一直停留在“排队中”的问题，暂停后会等待当前上传任务收口并释放队列槽，再允许继续正常执行。
4. 优化任务中心触发逻辑：传输任务只在首次创建时自动打开任务中心，后续进度消息不再反复弹出。
5. 优化上传和下载临时文件处理：上传先写入远端 `.download` 临时文件，下载先写入本地 `.download` 临时文件，传输完成后再重命名为原文件名。
6. 优化任务中心上传组展示，新增组内完成项数、总项数和当前处理子项提示，便于查看文件夹上传进度。

## v1.1.4

- 新增终端复制、粘贴快捷键 `Ctrl+Shift+C` / `Ctrl+Shift+V`（macOS 为 `Cmd+Shift+C` / `Cmd+Shift+V`），分别用于复制终端选区和把剪贴板文本粘贴进当前终端。
- 在设置弹窗的快捷键列表中新增终端复制、粘贴对应条目。
- 增加系统托盘

## v1.1.3

1. 修复 SFTP 新建文件、文件夹后右键菜单未关闭的问题，操作完成后菜单会自动收起。
2. 修复 SFTP 重命名和新建名称编辑时 `Ctrl+A` 误触发文件列表全选的问题，现在会优先全选输入框内的文件名。
3. 优化 macOS 远端系统状态采集，兼容 BSD `df`、`vm_stat` 和 macOS shell 输出差异，减少系统信息探测失败。
4. 优化 SSH 终端中文输入输出，使用 UTF-8 流式解码并通过 SSH 环境变量请求设置 UTF-8 locale，避免中文字符乱码。
5. 优化 macOS SSH 终端初始化方式，移除会污染 history 的交互式长命令，改用非交互路径同步和环境变量传递。
6. 优化 macOS 终端目录颜色支持，通过 SSH 环境变量请求启用 `CLICOLOR`，在服务器允许环境变量传递时恢复 `ls` 颜色。

## v1.1.2

- 新增默认本地终端，启动应用后会自动打开“本地”标签，Windows 默认进入 `C:\`，Linux 和 macOS 默认进入用户主目录。
- 新增本地终端交互能力，本地标签支持输入、窗口尺寸同步、关闭、重连、系统状态展示和 AI 命令执行。

## v1.1.1

- 优化 AI 回复结束后的输入体验，AI 输出完成后会自动聚焦到底部输入框，便于继续追问。
- 修复 macOS 中文输入法组词或选词时按回车会误触发 AI 发送的问题，现在回车会优先用于确认候选词。
- 修复 SSH 断开后点击重新连接总是报错的问题，断开清理旧会话后可通过原服务器信息恢复连接。
- 修复新建远程文件或文件夹后名称输入框未自动聚焦的问题，现在会自动聚焦并全选占位名称，便于直接改名。

## v1.1.0

- 新增 AI 助手面板，支持按终端标签页隔离对话，并结合当前服务器、终端路径、SFTP 路径、连接状态和近期终端输出提供上下文问答。
- 新增 AI Agent 诊断流程，支持 OpenAI 兼容模型的流式回复、Markdown 实时渲染、命令执行过程卡片和多轮命令结果续问。
- 新增 AI 权限模式：每次询问、自动审批、完全访问；内置只读命令白名单、高风险命令强制审批和 5 分钟审批有效期，降低误执行风险。
- 新增 AI 设置页，支持启用开关、多模型配置管理、当前模型切换、默认模式设置和 API Key 本地脱敏展示。
- 优化 AI 服务异常与未配置场景，AI 未启用或模型配置不完整时提供本地基础建议，网络或接口异常时返回更明确的错误提示。
- 优化终端与 SFTP 断开、重连体验，SSH 断开后自动清空陈旧 SFTP 列表并展示断开状态，终端重连成功后自动恢复主 SFTP 会话。
- 新增终端重连能力，修复旧连接事件影响新会话的问题，减少断线重连后的状态错乱。
- 优化文件交互基础能力，统一主 SFTP 与文件传输弹窗的选择、全选、范围选择和框选规则，避免删除中项目或上级目录虚拟项被错误选中。
- 增强终端、SFTP 和系统状态相关操作的参数校验与会话归属校验，减少非法参数或跨窗口调用导致的异常。
- 优化应用日志安全性，自动脱敏 password、API Key、token、secret、private key 等敏感字段。
- 新增 AI Agent 执行逻辑文档，说明对话链路、命令策略、审批流程和安全边界。

## v1.0.2

- 优化 SFTP 面板和文件传输弹窗的文件选择交互，支持类似 Windows 的框选、点击空白处取消当前选中项，并支持将文件拖拽移动到 `..` 上一级目录。
- 修复右键菜单和浮层菜单在窗口边缘时的位置异常，避免菜单越界或显示错位。
- 文件传输弹窗支持左右两侧选择同一个连接；同连接内传输会按复制处理，保留源文件不变。
- 优化文件传输体验，创建传输任务后不再自动关闭文件传输弹窗。
- 文件传输任务完成后自动刷新目标侧文件列表。
- 新增 SFTP 面板和文件传输弹窗内的拖拽移动操作，支持单选或多选文件、文件夹拖入同目录下其他文件夹，并在移动前弹出确认提示。
- 抽取远程文件列表公共组件，SFTP 面板和文件传输弹窗复用同一套文件行渲染、重命名和拖拽交互。
- 移除 SFTP 树形展示模式和相关设置项，仅保留当前层文件列表展示。
- 传输任务产生后自动展开任务列表，并提升任务列表层级，确保其优先显示在其他浮层之上。
- 统一将“数据传输”入口和提示文案调整为“文件传输”。

## v1.0.1

- 新增 SSH/SFTP 保活间隔设置，默认 10 秒，可在设置中调整，支持设置为 0 关闭保活。
- 新增空闲断开连接设置，默认 5 分钟；终端和主 SFTP 会话长时间无活动后会自动断开，支持设置为 0 关闭自动断开。
- 优化 SFTP 连接稳定性，主 SFTP、上传、下载、远程传输等连接都会应用保活配置。
- 新增 macOS 打包脚本 `dist-mac`，用于构建 macOS 安装包。
- 优化 macOS 窗口体验，新增左侧红黄绿窗口控制按钮，并接入 macOS 原生菜单栏。
- 新增 macOS 原生菜单中的设置、关于、文件传输、编辑和全屏相关操作。
- 优化文件编辑器撤销、重做逻辑，使 macOS 原生编辑菜单可作用于当前编辑器。
- 优化 SFTP 上传体验，支持在文件列表空白区域右键上传文件或文件夹到当前目录。
- 新增 SFTP 传输队列，同一时间默认最多执行 1 个上传、下载或远程传输任务。
- 新增关于弹窗，展示当前版本和项目 Gitee 地址。
- 优化应用图标生成流程，统一生成圆角图标，并用于 Windows、macOS 和应用顶部展示。
- 暂时隐藏 macOS 检查更新入口，避免未完成签名配置时触发异常更新流程。
